## Intentional Changes from Migration

The following changes were intentionally made during the typography system migration:

1. **Golden Ratio Type Scale**: All font sizes now follow the golden ratio (1.618) for harmonious proportions
2. **Responsive Typography**: Desktop sizes scale up by 1.125-1.25× from mobile base sizes
3. **Consistent Line Heights**: Line heights are based on golden ratio for optimal readability
4. **Semantic Font Weights**: Clear hierarchy using 400 (normal), 500 (medium), 600 (semibold), 700 (bold)
5. **Touch Target Compliance**: All interactive text elements meet 44×44px minimum on mobile
6. **WCAG AA Compliance**: All text meets contrast ratio requirements
